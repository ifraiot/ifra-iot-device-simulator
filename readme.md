# IFRA MQTT NODEJS

IFRA MQTT Nodejs test is a library for simulate data send  to ifra platform  http://iot.ifra.io.

## Installation

Use the package manager [MqttJS](https://www.npmjs.com/package/mqtt) to install.

```bash
npm install mqtt --save
```

## Usage

```javascript
 node index.js
```
